Coretact

Manage and share meshcore contact advertisements within your Discord servers. Each server becomes a separate mesh of contacts for your community.

Add to Discord

Easy Sharing

Share your meshcore contacts with a simple command. Add contacts directly from meshcore:// URLs.

Powerful Search

Find contacts by name, type, or key prefix. Filter companions, repeaters, and room devices.

Bulk Export

Download all contacts from your server as a single file for easy import into your devices.

Developer API

Access contacts programmatically via our REST API. All endpoints are available under /api/v1/

GET /api/v1/meshes
GET /api/v1/meshes/{'{server_id}'}
GET /api/v1/meshes/{'{server_id}'}/contacts
GET /api/v1/meshes/{'{server_id}'}/contacts/{'{public_key}'}
